- auth
- profile

### Things Cool in Kamu
- Full nextjs server actions
- custom tailwind plugins
- server authentication


### Things not yet cool in Kamu
- image & gender no API
- no onchange validation on input
- no animation yet

### How to run

- add .env.local and necessary env (see .env.examples)
- install and run
```bash
  pnpm install && pnpm dev
```

* node 20+ required
